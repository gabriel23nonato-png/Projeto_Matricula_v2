# Sistema de Matrículas Escolares
## E.M. Maria de Lourdes

### Criador
Gabriel Nonato Freitas do Carmo

### Objetivo
Sistema web local para matrícula de alunos da rede pública municipal.

### Público-alvo
Secretaria escolar e responsáveis legais.
Interface simples, family friendly, uso em rede local (Wi-Fi).

---

## Tecnologias Utilizadas
- Python 3.x
- Flask
- SQLite
- HTML + CSS
- Upload de arquivos organizados por aluno
- Exportação CSV/XLSX
- Geração de EXE com PyInstaller

---

## Funcionalidades Implementadas
- Cadastro de matrícula
- Upload de documentos:
  - Certidão de nascimento
  - RG/CPF da criança
  - RG dos responsáveis
  - Cartão SUS
  - Comprovante de residência
- Organização dos arquivos por aluno
- Listagem com busca
- Edição e exclusão de matrícula
- Ficha individual com impressão
- Uso em rede local
- Banco de dados editável
